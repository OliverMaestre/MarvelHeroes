# MarvelHeroes
Marvel Heroes Api test 


## Important

To consume the app, you only need a Marvel Public and Private Key and add it into the [apikey.properties ](https://github.com/DavidUps/Marvel/blob/1.0/apikeys.properties) like

    PUBLIC_KEY=""
    PRIVATE_KEY=""
